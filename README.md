# Pocket Legal AI

A lightweight AI-powered legal assistant built for everyday Nigerians to understand, question, and protect themselves from complex legal documents without hiring a lawyer.

## What it does
- Scans contracts and labels every clause as Fair, Risky, or Illegal
- Explains each clause in Plain English or Nigerian Pidgin
- Answers legal questions grounded in Nigerian law
- Works on any device with a browser

## Tech Stack
- HTML + Tailwind CSS (via CDN)
- PDF.js for PDF text extraction
- OpenRouter API (openrouter/auto) for AI reasoning
- localStorage for offline result caching
- Deployed on Vercel

## How to run locally
1. Clone this repo
2. Open index.html in your browser or use Live Server in VS Code
3. Add your OpenRouter API key in scanner.html and qa.html
